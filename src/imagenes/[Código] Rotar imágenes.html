[Código] Rotar imágenes

 por Lamashta el Mar 20 Ene 2009 - 14:56

Rotar imágenes


Para que no quede mal, se recomienda que todas las imágenes sean del mismo tamaño.


Lo colocamos en el mensaje del índice
Panel de administración > Visualización > Índice > General > Contenido del mensaje

<html>
<head>
<script language="Javascript" type="text/javascript">
mis_imagenes = new Array("IMAGEN","IMAGEN","IMAGEN")
mi_imagen = 0
imgCt = mis_imagenes.length
function rotacion() {
if (document.images) {
mi_imagen++
if (mi_imagen == imgCt) {
mi_imagen = 0
}
document.anuncio.src=mis_imagenes[mi_imagen]
setTimeout("rotacion()", 5 * 1000)
}
}
</script>
</head>
<body onload="rotacion()">
<img src="http://illiweb.com/fa/empty.gif" name="anuncio" alt="Anuncios" />
</body>
</html>

Lo que está en rojo lo sustituyes por la URL directa de la imagen, si quieres que salgan más imágenes sigues la serie ahí mismo, es decir que le añades otra ,"IMAGEN" dentro del paréntesis.
Obviamente colocando la url en el lugar de IMAGEN
Lo que está en azul son los segundos que tarda en cambiar de una imagen a otra.

También podemos colocarlo en un módulo personalizado
Pero para ello hay que seguir pasos distintos.
Primero vamos a crear una página HTML que contenga el código.
Panel de administración > Módulos > Gestión de páginas HTML
Le damos a "Crear página HTML" y rellenamos el formulario de este modo:
Título *: Rotar
¿Deseas utilizar la parte alta y la parte baja de la página de tu foro? NO
¿Utilizar esta página como índice? NO
Código HTML * : Aquí pegamos el código anterior.
Le damos a "Aceptar", volvemos a la gestión de páginas HTML y veremos que en la lista se encuentra la nueva página, copiamos la dirección de la página que hemos creado.
Ahora lo colocaremos en nuestro módulo personalizado:
En un módulo personalizado para nuestro foro
Panel de administración > Módulos > Gestión de los widgets del foro
En un módulo personalizado para nuestro portal
Panel de administración > Módulos > Gestión del portal > Estructura
Le damos a "Crear un módulo personalizado" y rellenamos las opciones de este modo:
Las tres primeras opciones las puedes rellenar como quieras, pero yo lo haré así.
Nombre del módulo: Rotar
Usar una tabla tipo: No
Título del módulo: Aquí no pongo nada
Fuente del módulo*: Aquí colocamos el siguiente código
<iframe src="URL" width="300" height="300" frameborder="0" scrolling="no"></iframe>
Lo que está en rojo es donde colocamos la dirección de la página HTML que hemos creado antes.
Lo que está en verde es el tamaño que tendrá la ventana que nos mostrará las imágenes, debemos adaptarla según el tamaño de nuestras imágenes.
Lo que está en morado es el grosor del borde que tendrá la ventana que mostrará las imágenes, si lo ponemos a "0" no tendrá borde.
Lo que está en rosa es para que se muestren o no barras de desplazamiento en la ventana que mostrará las imágenes, es mejor si lo dejáis en "no".

Le damos a registrar, colocamos nuestro módulo personalizado en la estructura de nuestro foro o portal y listo! ^.^
